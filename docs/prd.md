# وثيقة متطلبات تطبيق توليد الصور والفيديوهات بالذكاء الاصطناعي

## 1. نظرة عامة على التطبيق

### 1.1 اسم التطبيق
مولد الصور والفيديوهات بالذكاء الاصطناعي

### 1.2 وصف التطبيق
تطبيق أداة يتيح للمستخدمين توليد الصور والفيديوهات من خلال أوامر نصية باستخدام الذكاء الاصطناعي، مع إمكانية استخدام صورة مرجعية اختيارية، واختيار نسب العرض إلى الارتفاع المختلفة، وعرض سجل التوليد، وتحميل النتائج.

## 2. الوظائف الأساسية

### 2.1 صفحة توليد الصور (/images)
\n#### 2.1.1 واجهة المستخدم
- **قائمة منسدلة**: اختيار نموذج الذكاء الاصطناعي
  - nano banana pro
  - نماذج أخرى متاحة
- **حقل نصي**: إدخال الوصف النصي (Prompt) - إلزامي
- **حقل نصي**: إدخال الوصف السلبي (Negative prompt) - اختياري
- **قائمة منسدلة**: اختيار نسبة العرض إلى الارتفاع\n  - مربع (1:1)
  - أفقي (16:9)
  - عمودي (9:16)
  - مخصص (مع إدخال العرض والارتفاع)
- **زر رفع**: رفع صورة مرجعية - اختياري
- **شريط تمرير**: قوة المرجع (0-1) - يظهر فقط عند رفع صورة مرجعية
- **زر**: توليد الصورة\n
#### 2.1.2 عرض حالة التوليد
- عرض حالة المهمة: في الانتظار / قيد التنفيذ / نجح / فشل
- عند الانتهاء:
  - عرض الصورة النهائية
  - زر: تحميل الصورة
  - زر: نسخ الوصف النصي
  - زر: إعادة التوليد
\n#### 2.1.3 قسم توليد الصور المصغرة (Thumbnail)
- **حقل نصي**: إدخال الوصف النصي للصورة المصغرة - إلزامي
- **قائمة منسدلة**: اختيار نسبة العرض إلى الارتفاع\n  - مربع (1:1)
  - أفقي (16:9)
  - عمودي (9:16)
  - مخصص
- **زر**: توليد الصورة المصغرة
- عرض النتيجة:\n  - عرض الصورة المصغرة
  - زر: تحميل الصورة المصغرة
\n### 2.2 صفحة توليد الفيديوهات (/videos)

#### 2.2.1 واجهة المستخدم
- **قائمة منسدلة**: اختيار نموذج الذكاء الاصطناعي\n  - نموذج ذكاء اصطناعي مجاني لتوليد فيديوهات احترافية
  - نماذج أخرى متاحة
- **حقل نصي**: إدخال الوصف النصي (Prompt) - إلزامي
- **قائمة منسدلة**: اختيار نسبة العرض إلى الارتفاع\n  - مربع (1:1)
  - أفقي (16:9)
  - عمودي (9:16)
  - قصة (9:16)
  - مخصص
- **قائمة منسدلة**: مدة الفيديو
  - 4 ثوانٍ
  - 6 ثوانٍ\n  - 8 ثوانٍ
- **زر رفع**: رفع صورة بداية - اختياري (إذا كان المزود يدعم تحويل الصورة إلى فيديو)
- **زر**: توليد الفيديو

#### 2.2.2 عرض النتيجة
- عند الانتهاء:\n  - عرض صورة مصغرة
  - مشغل فيديو
  - زر: تحميل الفيديو

### 2.3 قسم سجل التوليد

#### 2.3.1 عرض السجل
- عرض آخر 50 عملية توليد لكل مستخدم (الصور والفيديوهات بشكل منفصل)
- كل عنصر يعرض:
  - معاينة
  - تاريخ الإنشاء
  - الحجم
  - الحالة
  - اسم المزود
- الإجراءات المتاحة:
  - عرض
  - تحميل
  - حذف\n
### 2.4 صفحة الإعدادات (/settings)

#### 2.4.1 إعدادات العرض
- **زر تبديل**: الوضع الليلي والنهاري
  - حفظ اختيار المستخدم
  - تطبيق الألوان المناسبة لكل وضع تلقائياً
- **قائمة منسدلة**: اختيار اللغة\n  - العربية
  - الإنجليزية
  - حفظ اختيار المستخدم
  - تطبيق اللغة المختارة على جميع عناصر الواجهة

#### 2.4.2 إعدادات إضافية
- إعدادات أخرى حسب احتياجات التطبيق
\n### 2.5 دعم اللغات المتعددة
- دعم اللغة العربية:\n  - واجهة مستخدم كاملة باللغة العربية\n  - تخطيط من اليمين إلى اليسار (RTL)\n  - جميع التسميات والنصوص بالعربية
- دعم اللغة الإنجليزية:
  - واجهة مستخدم كاملة باللغة الإنجليزية
  - تخطيط من اليسار إلى اليمين (LTR)
  - جميع التسميات والنصوص بالإنجليزية
- إمكانية التبديل بين اللغات من صفحة الإعدادات

## 3. الأدوات والنماذج المستخدمة
- **توليد الصور**: أداة nano banana pro ونماذج أخرى قابلة للاختيار
- **توليد الفيديوهات**: نموذج ذكاء اصطناعي مجاني لتوليد فيديوهات احترافية ونماذج أخرى قابلة للاختيار

## 4. متطلبات إضافية
- واجهة مستخدم بسيطة وسهلة الاستخدام
- استجابة سريعة وتجربة مستخدم سلسة
