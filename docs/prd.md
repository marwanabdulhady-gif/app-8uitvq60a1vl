# وثيقة متطلبات تطبيق توليد الصور والفيديوهات بالذكاء الاصطناعي

## 1. نظرة عامة على التطبيق

### 1.1 اسم التطبيق
مولد الصور والفيديوهات بالذكاء الاصطناعي

### 1.2 وصف التطبيق
تطبيق أداة يتيح للمستخدمين توليد الصور والفيديوهات من خلال أوامر نصية باستخدام الذكاء الاصطناعي، مع إمكانية استخدام صورة مرجعية اختيارية، واختيار نسب العرض إلى الارتفاع المختلفة، وعرض سجل التوليد، وتحميل النتائج.

## 2. الوظائف الأساسية

### 2.1 صفحة توليد الصور (/images)
\n#### 2.1.1 واجهة المستخدم
- **حقل نصي**: إدخال الوصف النصي (Prompt) - إلزامي
- **حقل نصي**: إدخال الوصف السلبي (Negative prompt) - اختياري
- **قائمة منسدلة**: اختيار نسبة العرض إلى الارتفاع\n  - مربع (1:1)
  - أفقي (16:9)
  - عمودي (9:16)
  - مخصص (مع إدخال العرض والارتفاع)
- **زر رفع**: رفع صورة مرجعية - اختياري
- **شريط تمرير**: قوة المرجع (0-1) - يظهر فقط عند رفع صورة مرجعية\n- **زر**: توليد الصورة\n\n#### 2.1.2 عرض حالة التوليد
- عرض حالة المهمة: في الانتظار / قيد التنفيذ / نجح / فشل
- عند الانتهاء:\n  - عرض الصورة النهائية
  - زر: تحميل الصورة\n  - زر: نسخ الوصف النصي
  - زر: إعادة التوليد

### 2.2 صفحة توليد الفيديوهات (/videos)

#### 2.2.1 واجهة المستخدم
- **حقل نصي**: إدخال الوصف النصي (Prompt) - إلزامي
- **قائمة منسدلة**: اختيار نسبة العرض إلى الارتفاع
  - مربع (1:1)
  - أفقي (16:9)
  - عمودي (9:16)
  - قصة (9:16)
  - مخصص
- **قائمة منسدلة**: مدة الفيديو\n  - 4 ثوانٍ
  - 6 ثوانٍ
  - 8 ثوانٍ\n- **زر رفع**: رفع صورة بداية - اختياري (إذا كان المزود يدعم تحويل الصورة إلى فيديو)
- **زر**: توليد الفيديو

#### 2.2.2 عرض النتيجة
- عند الانتهاء:
  - عرض صورة مصغرة
  - مشغل فيديو
  - زر: تحميل الفيديو
\n### 2.3 قسم سجل التوليد

#### 2.3.1 عرض السجل
- عرض آخر 50 عملية توليد لكل مستخدم (الصور والفيديوهات بشكل منفصل)\n- كل عنصر يعرض:\n  - معاينة
  - تاريخ الإنشاء
  - الحجم
  - الحالة
  - اسم المزود
- الإجراءات المتاحة:
  - عرض
  - تحميل
  - حذف
\n### 2.4 الوضع الليلي والنهاري
- زر تبديل بين الوضع الليلي والنهاري
- حفظ اختيار المستخدم
- تطبيق الألوان المناسبة لكل وضع تلقائياً
\n### 2.5 دعم اللغة العربية
- واجهة مستخدم كاملة باللغة العربية
- تخطيط من اليمين إلى اليسار (RTL)
- جميع التسميات والنصوص بالعربية

## 3. الأدوات والنماذج المستخدمة
- **توليد الصور**: أداة nano banana pro
- **توليد الفيديوهات**: نموذج ذكاء اصطناعي مجاني لتوليد فيديوهات احترافية
\n## 4. متطلبات إضافية
- واجهة مستخدم بسيطة وسهلة الاستخدام
- استجابة سريعة وتجربة مستخدم سلسة
